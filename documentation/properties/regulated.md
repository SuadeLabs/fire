# regulated

---

# loan

Parameter to define whether the loan product is regulated or unregulated.


# collateral

 Parameter to define whether the collateral qualifies the exposure as 'regulatory' for credit risk purposes

 In the UK, an exposure is classified as 'regulatory real estate' only if the collateral meets all criteria set out in Appendix 2 of the PRA Rulebook, Article 124A.
