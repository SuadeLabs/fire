# securitisation_id


---

Code used for the legal registration of a securitisation transaction or, if not available, the name by which the securitisation transaction is known in the market, or within the institution in case of an internal or private securitisation. Where the International Securities Identification Number -ISIN- is available (i.e. for public transactions), the characters that are common to all tranches of the securitisation shall be reported in this attribute. Also used for all tranches linked to the same transaction
