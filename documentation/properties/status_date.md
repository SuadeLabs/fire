# status_date

---

The **status_date** represents the date on which the security **status** was executed in the security schema. In the loan schema, it represents the date on which the status changed.

It is in date-time format in accordance with the ISO 8601 standard (YYYY-MM-DDTHH:MM:SSZ).

See also [**status**]
